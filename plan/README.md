# Exploratory Testing Assistant - Plan Index

This folder captures the hackathon MVP plan as small, focused docs.

## Files
- `plan/01-problem-and-goals.md` - problem statement, goals, non-goals, users.
- `plan/02-requirements-and-decisions.md` - functional + non-functional requirements and locked decisions.
- `plan/03-architecture-and-dataflow.md` - system boundaries, components, and data flow diagram.
- `plan/04-data-model.md` - Postgres schema outline and key entities.
- `plan/05-api.md` - API endpoints and payload shapes.
- `plan/06-extension.md` - Chrome extension design and capture pipeline.
- `plan/07-webapp.md` - web app UI and playback experience.
- `plan/08-ai-orchestration.md` - ADK multi-agent analysis workflow.
- `plan/09-milestones-and-risks.md` - delivery milestones, risks, and mitigations.
- `plan/10-hackathon-compliance.md` - Gemini 3 hackathon submission checklist.
- `plan/11-demo-script.md` - 3-minute demo script (Marathon Agent focus).

## Update policy
- Keep changes additive and minimal during the hackathon.
- Record decisions in `plan/02-requirements-and-decisions.md`.
- If scope changes, update the architecture and data model first.
