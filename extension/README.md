# QA Assist Chrome Extension

Load `extension/` as an unpacked extension in Chrome.

Features:
- Start/stop recording
- Active-tab video capture via offscreen document
- Console and network log capture via Debugger API
- Interaction capture from content script
- Marker shortcut: Ctrl+Shift+M (Cmd+Shift+M on macOS)
- Annotation shortcut: Ctrl+Shift+K (Cmd+Shift+K on macOS)

Configure API URL in popup before recording.
